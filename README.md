# Aesthetic Pomodoro Focus

Aplicativo de produtividade estilo Pomodoro com temas e gamificação.

**Nota:** Este repositório contém o código fonte de uma versão demo/backup.

## Funcionalidades

- Timer de Foco (Pomodoro)
- Sistema de XP e Níveis
- Loja de Temas e Sons
- Suporte a Inglês e Português

## Como Rodar

1. Instalar dependências:
   `npm install`

2. Rodar em desenvolvimento:
   `npm run electron:dev`

3. Gerar executável (Windows):
   `npm run dist:win`
